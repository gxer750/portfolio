Overview:

Lake Tahoe is a website for booking resorts and activities in the Lake Tahoe, California area. The site also provides tips and tricks for new visitors to better their experience. Their main objective is to drive visitors and campers to Lake Tahoe during vacations and holidays.

Visit Project View Code

Technologies Used:

  • Semantic HTML5
  • Responsive CSS3
  • Font Awesome
  • Interactive JavaScript
  • Wow JavaScript Plugin
  • Smooth Scroll JavaScript Plugin

Wow and Smooth Scroll were implemented into the project to bring Jquery like features without the library's heavy footprint.
